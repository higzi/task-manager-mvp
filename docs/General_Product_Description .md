# 2. Общее описание продукта

---

## 2.1 Кратко
**Task Manager MVP** — это веб-приложение для управления задачами, позволяющее пользователям создавать, редактировать и отслеживать задачи с использованием клиент-серверной архитектуры.

Приложение обеспечивает полный базовый цикл работы с задачами: **от создания до завершения**.

---

## 2.2 Какие проблемы решает

Приложение помогает решить следующие проблемы:

- Отсутствие централизованного инструмента для управления задачами  
- Сложность отслеживания прогресса выполнения задач  
- Потеря важных задач и дедлайнов  
- Использование разрозненных инструментов (заметки, мессенджеры и т.д.)  
- Снижение продуктивности из-за отсутствия структуры  

---

## 2.3 Для кого оно создано

Приложение предназначено для:

- Индивидуальных пользователей — личное планирование  
- Студентов — контроль заданий и дедлайнов  
- Разработчиков — ведение задач по проектам  
- Небольших команд — простое управление задачами  

---

## 2.4 В каких сценариях используется

Основные сценарии использования:

- Планирование ежедневных задач (to-do list)  
- Управление небольшими проектами  
- Отслеживание статусов задач (todo / in-progress / done)  
- Приоритизация задач  
- Быстрое создание и редактирование задач  
- Просмотр текущего списка задач  

---

## 2.5 Общий обзор функциональности

Приложение состоит из frontend и backend частей и предоставляет базовый функционал управления задачами.

### Основные возможности

- Создание задач  
- Редактирование задач  
- Удаление задач  
- Просмотр списка задач  
- Управление статусами задач  
- Хранение данных на сервере  
- Взаимодействие через REST API  
- Обновление интерфейса после действий пользователя  

---

### Архитектура

Приложение построено по клиент-серверной архитектуре, где frontend отвечает за пользовательский интерфейс, backend — за обработку данных и бизнес-логику, а база данных — за хранение информации.

---

Данный функционал соответствует уровню MVP и может быть расширен в будущих версиях приложения.
